# MyWeatherApp
This weather app not use third party libraries and using:
- Kotlin
- Coroutine
- Room
- Live Data
- UnitTest

# To run all test case in unit tes
you should:
1 uninstall this app
2 run app again
3 search "Moscow" to see result
4 run unit test
